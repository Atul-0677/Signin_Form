# Flipkart Login & Registration System

## About This Project

This is a desktop application developed using Python and Tkinter. The project allows users to create an account and log in using their registered credentials. User information is stored in an SQLite database.

I built this project to improve my understanding of Python GUI development, database integration, form validation, and user authentication.

## Features

* User Registration (Sign Up)
* User Login (Sign In)
* SQLite Database Integration
* Email Validation
* Password Validation
* Success and Error Messages
* Interactive User Interface
* Flipkart-Inspired Design

## Technologies Used

* Python
* Tkinter
* SQLite
* Pillow (Image Processing)

## Project Structure

* SignUp.py – Registration page
* LoginPage.py – Login page
* database.py – Database operations
* Flipkart-logo1.jpg – Application logo
* form-icon.png – Window icon
